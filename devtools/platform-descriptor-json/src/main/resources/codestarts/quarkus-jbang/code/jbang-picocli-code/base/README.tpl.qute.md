## Greet the world!

```shell script
./jbang src/EntryCommand.java hello
```

## Say Goodbye!

```shell script
./jbang src/EntryCommand.java goodbye
```

## Show help

```shell script
./jbang src/EntryCommand.java -h
```

or

```shell script
./jbang src/EntryCommand.java --help
```

## Show version

```shell script
./jbang src/EntryCommand.java -V
```

or

```shell script
./jbang src/EntryCommand.java -h
```
